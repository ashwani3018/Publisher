Web scraping: the act of automatically downloading a web page's data and
              extracting very specific information from it. The extracted
              information can be stored pretty much anywhere (database, file, etc.).

Web crawling: the act of automatically downloading a web page's data,
              extracting the hyperlinks it contains and following them.
              The downloaded data is generally stored in an index or a database to make it easily searchable.
